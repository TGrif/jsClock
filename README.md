# jsClock

demo: http://direct-shoot.com/jsClock



##Usage

var clock = new jsClock(audioContext, element);




##API

	startClock()

	pauseClock()

	stopClock()

	clockIsRunning()

	clockTime()

	setTempo()

	setMesure()

	setMetronome()
